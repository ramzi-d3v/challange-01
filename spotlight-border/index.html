How the mask works

A ::before pseudo-element fills with a viewport-fixed radial gradient tracking --mx / --my. A CSS mask with mask-composite: exclude punches out the interior, leaving only the border ring lit up.

--spot-hsl: 74 100% 50%;  /* lime */
--spot-fade: 20vmin;      /* falloff radius */
--bw: 2px;                /* ring thickness */
